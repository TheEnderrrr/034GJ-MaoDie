## 📖 游戏简介

**点击球** 是一款独特的 2D 物理游戏。玩家控制一个基础小球，通过鼠标右键移动，左键与场景中不同颜色的目标球互动。每种颜色的球都拥有独特的能力，合理利用它们是获得高分的关键。是选择瞬间位移躲避危险，还是分裂增殖扩大势力？一切由你决定！

## 🕹️ 操作指南

| 鼠标按键 | 操作描述 |
| :---: | :--- |
| **右键 (按住)** | **移动模式**：按住右键，你的小球会持续向鼠标光标方向移动。松开即停止加速。 |
| **左键 (点击)** | **互动模式**：当准星对准场景中的彩色小球时，点击左键触发该颜色对应的特殊效果。 |

> 💡 **提示**：游戏考验反应速度与策略判断，请根据周围局势灵活选择点击目标！

## 🔮 核心机制：彩色小球图鉴

场景中会生成不同颜色的小球，点击它们将触发以下独特效果：

### 🟢 绿色小球：时空置换 (Teleport)
- **效果**：点击后，你的小球会与绿色小球**瞬间交换位置**。
- **策略**：用于快速脱离包围圈，或穿越障碍物到达地图另一端。

### 🔵 蓝色小球：细胞分裂 (Split)
- **效果**：点击后，该蓝色小球会**分裂成三个**独立的小球。
- **策略**：增加场景混乱度，可能制造新的机会或阻碍。

### 🟣 紫色小球：质量吞噬 (Absorb)
- **效果**：点击后，与目标建立连接并**持续吸收其质量**，直至对方完全消失。
- **风险**：如果连接中途被打断（如距离过远或被干扰），吸收过程失败，无法获得质量。
- **策略**：适合在安全环境下快速壮大自身体型。

### 🔴 红色小球：连锁爆炸 (Explosion)
- **效果**：点击后产生剧烈爆炸，**炸飞周围所有小球**并削减它们的质量。红色小球自身消失，化为随机飞散的特效。
- **策略**：解围神器！当被大量小球包围时使用，可瞬间清场并削弱敌人。

### 🟠 橙黄色小球：引力奇点 (Gravity Well)
- **效果**：点击后，该球质量快速减小，同时产生**强大引力**，将周围所有小球向中心聚拢。
- **限制**：若周围没有其他小球，则不会触发引力效果。
- **策略**：用于将分散的敌人聚集在一起，配合红色小球使用效果更佳。

### ⚫ 棕黑色小球：质量均分 (Equalize)
- **效果**：点击后，与周围所有小球建立连接，并将**自身质量平均分配**给它们。
- **策略**：高风险高回报。如果你体型巨大，这会削弱你；如果你体型较小，这可能让你瞬间变大。需谨慎判断局势。
